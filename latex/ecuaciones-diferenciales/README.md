# Fuente LaTeX de Ecuaciones Diferenciales

Esta carpeta contiene las clases editables del curso de Ecuaciones
Diferenciales publicado en CMSpec.

## Editar desde GitHub

1. Abre la clase que quieras modificar (`Clase_1.tex` a `Clase_11.tex`).
2. Presiona el ícono del lápiz, **Edit this file**.
3. Realiza la corrección y selecciona **Commit changes**.
4. Pide a Codex: **“Sincroniza Ecuaciones Diferenciales desde GitHub”**.

## Correspondencia con el sitio

- `Clase_1.tex`: Fundamentos y variables separables
- `Clase_2.tex`: Ecuaciones exactas y lineales
- `Clase_3.tex`: Factor integrante y ecuaciones homogéneas
- `Clase_4.tex`: Sustituciones y ecuaciones de Bernoulli
- `Clase_5.tex`: Ecuaciones de orden superior
- `Clase_6.tex`: Ecuaciones homogéneas con coeficientes constantes
- `Clase_7.tex`: Coeficientes indeterminados y variación de parámetros
- `Clase_8.tex`: Cauchy–Euler y transformada de Laplace
- `Clase_9.tex`: Resolución con Laplace y función de Heaviside
- `Clase_10.tex`: Traslación y sistemas diferenciales
- `Clase_11.tex`: Series de Fourier y problemas de frontera

`Clase_12.tex` queda conservada como material adicional y no se publica por
ahora. El comando `npm run course:sync:edo` convierte las once clases activas
con plasTeX y actualiza el contenido web.
